
# 見てて汚いリポジトリ

## 説明
このリポジトリは **目に優しくない** ことを目的として作られました。  
色、余白、アニメーション、古いタグ、無駄なJavaScript…全てがカオス。  
人間の目にダメージを与えることを第一に設計されています。

## 使い方
1. クローンする
```bash
git clone https://github.com/dev-qianyu/dev-qian.github.io.git
````

2. `ugly.html` をブラウザで開く
3. 目が痛くなるのを覚悟して見る

## 注意

* デザインの良心はゼロです
* SEOとかは適当にメタタグ入れてますが無駄です
* 古い `<marquee>` タグ使ってます
